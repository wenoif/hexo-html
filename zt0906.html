用户没有sudo权限

提示： user is not in the sudoers file. This incident will be reported.

解决：

  1. su切换到root用户

  2. vim /etc/sudoers

    找到这一行：root ALL=(ALL) ALL
    并在下面新添加一行：用户名 ALL=(ALL) ALL

  3. :wq!

  4. 切换到原来的用户
